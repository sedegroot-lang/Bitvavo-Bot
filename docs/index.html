Wat doet de bot?

Volledig geautomatiseerde crypto trading bot voor Bitvavo, speciaal gebouwd voor de Nederlandse markt. 100% gratis, open source, en draait op je eigen Windows PC — geen abonnement, geen cloud, geen data die ergens naartoe gaat.

📈

Trailing Stop Loss

Volgt de koers omhoog en verkoopt automatisch bij een instelbare daling. Pakt winst zonder handmatig ingrijpen.

📉

DCA — Dollar Cost Averaging

Koopt automatisch bij op instelbare dipniveaus. Spreidt je aankopen en verlaagt je gemiddelde aankoopprijs.

🤖

Grid Trading + XGBoost AI

Handelt op automatische koerslagen. De AI-module traint wekelijks op je eigen trade-geschiedenis en filtert slechte signalen.

📱

Telegram Notificaties

Ontvang een bericht bij elke order, trade of fout — direct op je telefoon via Telegram.

📊

Browser Dashboard

Volledig dashboard op localhost:5001 — portfolio, orders, winst/verlies, backtests en instellingen.

🔬

Backtesting

Test je strategie op echte historische Bitvavo candles vóór je live gaat. Geen gokwerk.

Installatie in 5 minuten

1

Download de bot

Download de nieuwste release als ZIP van GitHub en pak hem uit.

https://github.com/sedegroot-lang/Bitvavo-Bot/releases/latest
2

Dubbelklik setup.bat

Installeert Python-dependencies en begeleidt je door de configuratie stap voor stap.

3

Voer je Bitvavo API key in

Maak een API key aan op bitvavo.com met alleen trading-rechten (geen opname).

4

Start de bot

Dubbelklik start_automated.bat en open het dashboard op localhost:5001.
