================================================================
  BITVAVO BOT - VOLLEDIGE INSTALLATIE & GEBRUIK
  Voor portfolio van ~100 EUR (incl. grid bot)
================================================================

Deze bot is precies dezelfde als die mijn vriend draait:
  - Trailing trading bot (koopt + verkoopt automatisch)
  - Optioneel: Grid bot (handelt in vast prijsbereik)
  - AI Supervisor (LSTM + XGBoost ensemble)
  - Auto-backup, Auto-retrain (XGB wekelijks)
  - Dashboard met live overzicht
  - Health Monitor sidecar venster
  - Een klik = alles draait automatisch


================================================================
  STAP 0 - BELANGRIJK: NIET IN ONEDRIVE!
================================================================

Pak de ZIP NIET uit in OneDrive, Dropbox of Google Drive.
OneDrive synct realtime en zet je config terug naar oudere versies.

GOED:  C:\BitvavoBot\   of   D:\Trading\BitvavoBot\
FOUT:  C:\Users\<naam>\OneDrive\...
FOUT:  C:\Users\<naam>\Documents\...  (als Documents in OneDrive zit)


================================================================
  STAP 1 - Pak de ZIP uit
================================================================

Pak BitvavoBot_share.zip uit naar bv. C:\BitvavoBot\


================================================================
  STAP 2 - Python 3.11 of 3.12 installeren
================================================================

Download van https://python.org/downloads
BELANGRIJK: vink "Add Python to PATH" aan tijdens installatie!


================================================================
  STAP 3 - Setup script draaien (een klik!)
================================================================

In de map C:\BitvavoBot\ vind je:   setup.bat

DUBBELKLIK setup.bat

Dit doet automatisch:
  - Maakt de virtual environment (.venv) aan
  - Installeert alle Python packages (xgboost, pandas, flask, etc.)
  - Klaar in 3-5 minuten

Als xgboost of lightgbm faalt: open PowerShell in de map en draai:
   .\.venv\Scripts\python.exe -m pip install -r requirements.txt


================================================================
  STAP 4 - Bitvavo API keys aanmaken
================================================================

1. Ga naar https://account.bitvavo.com/user/api
2. Maak een nieuwe key aan
3. Geef ALLEEN: [v] View info  +  [v] Trade
4. NOOIT [v] Withdrawals aanvinken!
5. IP whitelist op je eigen IP (sterk aanbevolen)


================================================================
  STAP 5 - .env bestand aanmaken
================================================================

In de map C:\BitvavoBot\ ligt al .env.example
Kopieer dit bestand en hernoem de kopie naar:   .env

Open .env in notepad en vul in:

   BITVAVO_API_KEY=jouw_eigen_key
   BITVAVO_API_SECRET=jouw_eigen_secret
   TELEGRAM_BOT_TOKEN=
   TELEGRAM_CHAT_ID=

Telegram velden mag je leeg laten (tenzij je meldingen wilt).

================================================================
  STAP 9 - GRID BOT INSCHAKELEN (voor je 100 EUR portfolio)
================================================================

Wacht eerst 1-2 dagen tot trailing bot stabiel draait. Dan grid aanzetten:

1. Stop de bot (Ctrl+C in hoofdvenster).

2. Open je local config:
   notepad "$env:LOCALAPPDATA\BotConfig\bot_config_local.json"

3. Zoek "GRID_TRADING" en wijzig:
       "enabled": false   ->   "enabled": true

   Met de instellingen die ik je gaf:
   - 30 EUR budget (zie BUDGET_RESERVATION.grid_bot_max_eur)
   - ETH-EUR als markt (rustige coin, perfect voor grid)
   - 8 grids (levels)
   - 12% stop-loss als prijs uit bereik valt
   - 10% take-profit per gerealiseerde fill

4. Sla op en herstart: dubbelklik start_automated.bat

5. Check dashboard, tab "Grid":
   - Actieve buy/sell orders per level
   - Gerealiseerde winst per fill
   - Of prijs binnen het bereik valt

WAARSCHUWINGEN GRID:
  - Bereik te smal -> prijs valt eruit, grid stopt -> coins blijven vast
  - Bereik te breed -> weinig fills, lage winst
  - GEBRUIK ALLEEN ETH/BTC/SOL voor grid, NOOIT memecoins
  - Begin klein (30 EUR is prima voor 100 EUR portfolio)
  - Houd 1-2 weken in de gaten voor je opschaalt

================================================================
  VEELVOORKOMENDE PROBLEMEN
================================================================

start_automated.bat sluit direct
  -> .venv niet aanwezig. Eerst setup.bat draaien.

Bot doet niks / koopt niet
  -> EUR saldo voldoende? (minstens 70 EUR vrij voor trailing)
  -> MIN_SCORE_TO_BUY = 8 is streng (correct, niet verlagen!)
  -> Wacht: signalen zijn schaars, soms uren niets

Config wijzigt niks
  -> Edit je echt %LOCALAPPDATA%\BotConfig\bot_config_local.json?
     notepad "$env:LOCALAPPDATA\BotConfig\bot_config_local.json"

Module not found / ImportError
  -> .\.venv\Scripts\python.exe -m pip install -r requirements.txt

AI / XGBoost errors bij start
  -> Tijdelijk uitzetten in local config:
     "USE_LSTM": false,
     "LLM_SUPERVISOR_ENABLED": false

Config wordt elke dag teruggezet
  -> Bot staat in OneDrive. Verplaats naar C:\BitvavoBot\

Heartbeat stale alerts
  -> Al uit (HEARTBEAT_STALE_ALERT_ENABLED = false in jouw config)
