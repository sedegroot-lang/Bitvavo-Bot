📱 Telegram Notificaties

Ontvang real-time meldingen op je telefoon bij elke koop, verkoop en waarschuwing.

Aanmaken via @BotFather op Telegram
Ophalen via api.telegram.org/bot{TOKEN}/getUpdates
📖 Hoe stel ik Telegram in?
  1. Open Telegram → zoek @BotFather → stuur /newbot
  2. Geef een naam en gebruikersnaam (eindigt op bot)
  3. Kopieer het token dat je krijgt en plak het hierboven
  4. Stuur je nieuwe bot een willekeurig berichtje
  5. Ga naar api.telegram.org/bot{TOKEN}/getUpdates
  6. Kopieer het getal achter "id": — dat is je Chat ID
